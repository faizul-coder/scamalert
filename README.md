# ScamAlert Selangor v0.7.6

Prototaip aplikasi web amaran awal untuk menyemak mesej mencurigakan.

## Ciri utama
- Satu halaman semakan mesej
- Skor risiko dalam bentuk meter
- Analisis lakuan pertuturan
- Analisis emosi
- Frasa dikesan
- Cadangan tindakan selamat
- Paparan bersih tanpa background design
- Semua kotak utama mempunyai outline hitam nipis

## Cara jalankan
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Fail utama
- `app.py`
- `requirements.txt`
- `.streamlit/config.toml`
