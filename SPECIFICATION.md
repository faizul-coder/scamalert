# SPECIFICATION

Nama aplikasi: ScamAlert  
Versi: v0.8.1 Integrated Friendly Labels  
Jenis: Prototaip aplikasi web amaran awal

## Fokus utama
ScamAlert menganalisis mesej mencurigakan melalui tiga lapisan:
1. **Makna tersurat dan makna tersirat**
2. **Pencetus emosi**
3. **Gerakan strategi penipuan**

## Penambahbaikan v0.8.1
- Nama aplikasi ditukar sepenuhnya kepada **ScamAlert**.
- Tiga kad analisis menggunakan istilah yang lebih mesra pengguna.
- Frasa seperti **“bayar dulu sebelum keluarkan duit”** diklasifikasikan sebagai berisiko tinggi kerana menandakan pola scam bayaran sebelum pengeluaran.
- Bahagian **Peta Gerakan Penipuan** memaparkan laluan strategi scam dalam bentuk langkah demi langkah.
- Bahagian hasil disusun kepada:
  1. Keputusan Keseluruhan
  2. Tiga Enjin Analisis
  3. Peta Gerakan Penipuan
  4. Frasa dan Petanda Dikesan
  5. Cadangan Tindakan Selamat
  6. Penafian
